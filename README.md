# Skill Library

A collection of thinking and working skills written in Markdown.  
Pick a skill, copy it, paste it into any AI — the AI reads the structure and applies it immediately.

---

## How it works

```
1. Pick a skill
2. Copy MD
3. Paste into any AI (Claude, ChatGPT, Gemini…)
4. AI applies it
```

No prompt engineering. No explanation needed.  
The skill is structured so the AI understands it directly.

---

## Why Markdown

Most knowledge sharing stops at "read → done."

Skill Library is different:

```
Read
↓
Copy
↓
AI reads the structure
↓
Apply
↓
Improve
↓
Share back
```

Markdown is the common language between humans and AI.  
A skill written in Markdown is readable by both.

## Usage example

**Mission: Run better meetings**

1. Open Skill Library
2. Pick `Meeting Decision Skill` + `Observation Before Opinion`
3. Copy both Markdowns
4. Paste into AI with your meeting notes
5. Output: decision log, action items, next owners

**Mission: Validate a new idea**

1. Pick `Concept Stress-Test`
2. Copy MD → paste into AI with your concept
3. Output: go / no-go decision with assumptions exposed

## Add your own skill

Click `+ Add Skill` in the UI, or write a Markdown file directly:

```markdown
# Skill Title
version: 0.1 | category: thinking | ai_readable: true

## Purpose
One sentence.

## When to use
Describe the situation.

## Steps
...
```
